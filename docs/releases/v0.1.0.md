# Engineering Ownership v0.1.0

The first public release provides a tool-neutral ownership workflow for
AI-assisted software development:

- R0–R3 risk-proportional evidence;
- current-diff-bound command verification;
- durable change records and architecturally significant ADRs;
- optional understanding gaps and revisit dates without a default quiz gate;
- eight evidence tags without scores;
- Codex and Claude Code plugin packaging;
- a Python 3.11+ standard-library CLI.

v0.1 intentionally excludes automatic hooks, MCP, telemetry, dashboards, PyPI,
and official marketplace submission.
